``requirements.txt``
====================

``requirements.txt`` は、 Python のパッケージマネージャである `pip <https://pip.pypa.io>`_ 用のパッケージ情報を記述したファイルです。

pip の仕組み上 ``requirements.txt`` という名前にしないといけない制約はありませんが、 `pip 公式のサンプルでこのファイル名が使われており <https://pip.pypa.io/en/stable/user_guide/?highlight=requirements.txt#requirements-files>`_ 、実際にも多くの Python プロジェクトが ``requirements.txt`` というファイルに依存パッケージ情報を記述しています。
