``composer.lock``
=================

``composer.lock`` は、 PHP のパッケージの依存関係管理ツールである `Composer <https://getcomposer.org/>`_ を使ってインストールされたパッケージの情報を格納するためのファイルです。

:doc:`composer.json` がインストールすべきパッケージの情報を含むさまざまな情報を格納するのに対して、こちらの ``composer.lock`` は実際にインストールされたパッケージの情報のみ格納します。 ``composer.lock`` は、複数の開発者が開発に関わるプロジェクトなどにおいて使用するパッケージのバージョンを厳密に合わせたい場合などに有用です。
