``package-lock.json``
=====================

``package-lock.json`` は、 `Node.js <https://nodejs.org/en/>`_ のパッケージ管理ツール `npm <https://www.npmjs.com/>`_ でインストールされたパッケージのバージョンや依存関係についての情報を記述したファイルです。

npm を使ってプロジェクト単位のパッケージを変更したときに自動的に生成されます。

* `package-lock.json | npm Documentation <https://docs.npmjs.com/files/package-lock.json>`_
